## Table of Contents

## Summary
Moss Control disables ability to:
- plant trees on moss
- convert moss into rooted dirt/podzol

## Installation
1. Build the plugin
```
git clone https://github.com/extstudios/MossControl.git
cd MossControl
mvn clean package
```
2. Drop into your server
